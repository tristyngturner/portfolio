Class re:union

In Class re:union, we teamed together to create a place for high school alumni to search their high school and classmates in order to get in touch, remember good times and potentially band together to plan a reunion. We used Sequelize to create and manage our own api and database, Bootstrap, CSS and Vanta.js for styling, Handlebars.js for HTML rendering, as well as other technologies.
